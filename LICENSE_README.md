# Odoo 19 Installer (Lite Online)

## راه‌اندازی سریع

1. فایل `start_ui_online.bat` را اجرا کنید
2. مرورگر به آدرس `http://127.0.0.1:5000` باز می‌شود
3. مراحل نصب را از UI دنبال کنید

## فعال‌سازی لایسنس

### روش ۱: فایل لایسنس (توصیه شده)
- بعد از خرید از سایت، فایل `license.oml` را دانلود کنید
- فایل را کنار `start_ui_online.bat` قرار دهید
- برنامه را اجرا کنید — لایسنس خودکار شناسایی می‌شود

### روش ۲: وارد کردن دستی
- در UI، بخش «License Management» را باز کنید
- کد لایسنس یا محتوای فایل `.oml` را paste کنید
- دکمه «فعال‌سازی» را بزنید
