معرفی OdooMaster

OdooMaster یک نصاب هوشمند و کامل برای راه‌اندازی Odoo در ویندوز است. این ابزار تمام مراحل پیچیده نصب را به یک فرآیند ساده و خودکار تبدیل می‌کند.

نصب OdooMaster

قبل از شروع نصب، مطمئن شوید که پیش‌نیازهای سیستم را دارید. برای اطلاعات بیشتر صفحه دانلود را ببینید.

دانلود نصاب

از صفحه دانلود، نسخه مناسب سیستم خود را دانلود کنید. برای نصب آفلاین، نسخه کامل را انتخاب کنید.

اجرای نصاب

فایل دانلود شده را با کلیک راست و "Run as Administrator" اجرا کنید تا دسترسی‌های لازم را داشته باشد.

انتخاب کامپوننت‌ها

در رابط نصاب، کامپوننت‌های مورد نظر را انتخاب کنید: Python، PostgreSQL، Git و Odoo.

شروع نصب

روی دکمه "نصب" کلیک کنید و منتظر بمانید. پیشرفت نصب به صورت زنده نمایش داده می‌شود.

فعال‌سازی لایسنس

پس از نصب، برای استفاده از امکانات کامل باید لایسنس خود را فعال کنید.

هر لایسنس فقط روی یک سیستم قابل فعال‌سازی است. اگر نیاز به انتقال دارید، ابتدا از پنل کاربری غیرفعال کنید.

مراحل فعال‌سازی

  1. وارد نرم‌افزار OdooMaster شوید
  2. از منوی "لایسنس"، گزینه "فعال‌سازی" را انتخاب کنید
  3. کد لایسنس ۲۵ کاراکتری خود را وارد کنید
  4. روی "فعال‌سازی" کلیک کنید

تنظیمات پایه

پس از نصب، فایل پیکربندی Odoo در مسیر زیر قرار دارد:

C:\OdooMaster\odoo19\odoo.conf

تنظیمات مهم

[options]
; پورت HTTP
http_port = 8069

; تنظیمات دیتابیس
db_host = localhost
db_port = 5432
db_user = odoo
db_password = your_password

; مسیر افزونه‌ها
addons_path = C:\OdooMaster\odoo19\addons

; لاگ
logfile = C:\OdooMaster\logs\odoo.log
log_level = info

; Workers (برای Production)
workers = 4
پس از تغییر تنظیمات، سرویس Odoo را ریستارت کنید تا تغییرات اعمال شوند.

مدیریت دیتابیس

PostgreSQL توسط OdooMaster به صورت خودکار نصب و پیکربندی می‌شود. برای مدیریت دیتابیس‌ها:

ایجاد دیتابیس جدید

  1. مرورگر را باز کنید و به http://localhost:8069 بروید
  2. روی "Manage Databases" کلیک کنید
  3. گزینه "Create" را انتخاب کنید
  4. نام دیتابیس و رمز عبور مدیر را وارد کنید

فارسی‌سازی کامل

برای فارسی‌سازی کامل Odoo، مراحل زیر را دنبال کنید:

  1. وارد Odoo شوید و به Settings بروید
  2. در قسمت Languages، روی "Add a Language" کلیک کنید
  3. زبان Persian (IR) / فارسی را انتخاب کنید
  4. صفحه را رفرش کنید
برای تجربه بهتر، افزونه‌های فارسی OdooMaster را نصب کنید که شامل تقویم شمسی و گزارشات فارسی است.

سوالات متداول

چگونه Odoo را ریستارت کنم؟

در OdooMaster روی دکمه "Restart" کلیک کنید، یا از Services ویندوز سرویس "OdooMaster" را ریستارت کنید.

پورت پیش‌فرض چیست؟

پورت پیش‌فرض HTTP عدد 8069 است. می‌توانید آن را در فایل تنظیمات تغییر دهید.

چگونه به نسخه جدید به‌روزرسانی کنم؟

از بخش "به‌روزرسانی" در نرم‌افزار یا دانلود نسخه جدید از سایت استفاده کنید. دیتابیس‌ها حفظ می‌شوند.
